# MCRedStoneModpack
## 目的
每次更新MC版本时都要去[Modrinth](https://modrinth.com)和[CurseForge](https://www.curseforge.com/minecraft/search?class=mc-mods&page=1&pageSize=20&sortType=1)东找西找mod,而生电模组加上优化模组,加起来动辄一百多个,非常麻烦,故建此项目简化该过程.

## 快捷键对照表

|精确放置|B|

|投影设置|M|
|材料列表|M+L|
|原理图验证|M+V|
|容器预览|Alt/⌥|
|tweakeroo|X+C|
|全部制作|Ctrl+A|
|全部丢弃|Ctrl ⌃+Shift ⇧+Q|
|显示配方|A|
|光照等级显示|F6|
|史莱姆区块显示|F7|
|图形渲染器|F8|
|区块边界|F9|
|高亮坐标|C|
